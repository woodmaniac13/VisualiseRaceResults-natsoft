1. scripts/natsoft-live-browser-scraper.py (live crawling, supported)
2. scripts/natsoft-parser.js (parse Result/Times HTML)

## Live Data Policy

For live crawling, use browser navigation from the Natsoft homepage.

The `reference/` folder is a learning and validation dataset only. It is not a runtime fallback source for live crawl failures.
